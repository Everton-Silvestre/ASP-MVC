TIPOS DE DADOS
---------------------
NM - NOMES
DT - DATAS
NR - NUMEROS
VL - VALORES(MOEDA)
DS - DESCRIÇÕES
ID - PARA CHAVES PRIMÁRIAS, EXCETO CHAVES COMPOSTAS, SOMENTE QUANDO UNICOS

TIPOS DE ESTRUTURA
---------------------
TB - TABELAS
SP - PROCEDURES
FN - FUNCTIONS
IX - INDICES
PK - CHAVES PRIMARIAS (PRIMARY KEY)
FK - CHAVES ESTRANFGEIRAS (FOREIGN KEY)

BOAS PRATICAS
---------------------

- TODAS AS NOMENCLATURAS EM LETRA MAIUSCULA
- NÃO COLOCAR NOMES DE TABELAS NO PLURAL
- SEPARAR PREFIXO E PALAVRAS COM UNDERSCORE '_'
